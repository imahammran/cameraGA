# cameraGA
Genetic Algorithm implementation in a recommender website

This website suggest the best combination of DSLR camera and its lens to use for user's preferred budget, photography style and also the location where the body and lens available.

Fitness function is higher when the combination price of the body and lens is les than user's entered budget and the location is similar to user's input. As for the photography style, the fitness is calculated based on the focal length (f/length) and f/stop of the lens that is suitable with the user's input style.
